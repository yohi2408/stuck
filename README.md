# 📈 מערכת ניתוח מניות והמלצות השקעה

מערכת מקיפה לניתוח מניות בזמן אמת עם המלצות השקעה חכמות המבוססות על ניתוח טכני ופונדמנטלי.

## ✨ תכונות

- **ניתוח טכני מתקדם**: Moving Averages (SMA 20/50), RSI, MACD, Bollinger Bands
- **ניתוח פונדמנטלי**: P/E Ratio, Market Cap, Beta, Dividend Yield
- **הערכת סיכונים**: ניתוח volatility, beta, ומגמות מחיר
- **המלצות חכמות**: המלצות לטווח קצר (1-3 חודשים) וארוך (6-12 חודשים)
- **ממשק משתמש מודרני**: עיצוב dark mode עם אנימציות וגרפים אינטראקטיביים
- **תמיכה ב-API כפול**: Alpha Vantage (ראשי) + Yahoo Finance (גיבוי)

## 📋 דרישות מערכת

- Python 3.8 ומעלה
- חיבור לאינטרנט
- דפדפן מודרני (Chrome, Firefox, Edge)

## 🚀 התקנה והרצה

### שלב 1: התקנת תלויות Python

פתח את ה-Command Prompt או PowerShell בתיקייה `c:\Users\maple\Downloads\interactive` והרץ:

```bash
pip install -r requirements.txt
```

### שלב 2: הרצת השרת

הרץ את השרת Flask:

```bash
python api_server.py
```

אתה אמור לראות הודעה:
```

📊 Server running on http://localhost:5000
```

### שלב 3: פתיחת הממשק

פתח את הקובץ `index.html` בדפדפן שלך:
- לחץ לחיצה כפולה על `index.html`, או
- גרור את הקובץ לחלון הדפדפן, או
- לחץ לחיצה ימנית ובחר "Open with" → הדפדפן שלך

## 📖 איך להשתמש

### חיפוש מניה בודדת

1. הכנס סימול מניה בשדה החיפוש (לדוגמה: `AAPL`, `MSFT`, `GOOGL`)
2. לחץ על "נתח מניה" או לחץ Enter
3. המערכת תציג:
   - **המלצות טווח קצר וארוך**
   - **רמת סיכון** עם הסבר מפורט
   - **ניתוח טכני**: מגמה ומומנטום
   - **ניתוח פונדמנטלי**: P/E, Market Cap, Beta
   - **גרף מחירים** עם ממוצעים נעים
   - **הסבר** מפורט להמלצה

### בחירות מהירות

לחץ על אחד הכפתורים המהירים (Apple, Microsoft, Google, וכו') לניתוח מיידי.

### המלצות מובילות

לחץ על "טען המלצות" כדי לקבל ניתוח של 5 המניות הפופולריות ביותר עם דירוג.

## 🎯 הבנת ההמלצות

### המלצות טווח קצר (1-3 חודשים)
- **קנייה חזקה** (Strong Buy): הזדמנות מצוינת לקנייה
- **קנייה** (Buy): מומלץ לקנות
- **החזקה** (Hold): להמשיך להחזיק או להמתין
- **מכירה** (Sell): מומלץ למכור
- **מכירה חזקה** (Strong Sell): מומלץ למכור בהקדם

### המלצות טווח ארוך (6-12 חודשים)
- **קנייה והחזקה חזקה**: מניה איכותית להשקעה ארוכת טווח
- **קנייה והחזקה**: מתאים להשקעה ארוכת טווח
- **החזקה**: להמשיך להחזיק
- **להימנע**: לא מומלץ להשקעה ארוכת טווח

### רמות סיכון
- **סיכון נמוך** 🟢: מניה יציבה עם תנודתיות נמוכה
- **סיכון בינוני** 🟡: תנודתיות סבירה
- **סיכון גבוה** 🔴: תנודתיות גבוהה, מתאים למשקיעים מנוסים

## 🔧 הגדרות מתקדמות

ניתן לערוך את `config.json` כדי להתאים את פרמטרי הניתוח:

```json
{
  "analysis_parameters": {
    "sma_short": 20,        // ממוצע נע קצר
    "sma_long": 50,         // ממוצע נע ארוך
    "rsi_period": 14,       // תקופת RSI
    "rsi_oversold": 30,     // רמת oversold
    "rsi_overbought": 70    // רמת overbought
  }
}
```

## 🔑 API Key

המערכת משתמשת ב-Alpha Vantage API. ה-API key שלך כבר מוגדר ב-`config.json`.

אם תרצה להחליף את ה-key:
1. קבל key חדש מ-https://www.alphavantage.co/support/#api-key
2. ערוך את `config.json` ושנה את `"api_key"`
